# AI Trading Analytics Dashboard

A Streamlit-based financial analytics dashboard for stocks, forex, and crypto markets.

## Features
- Candlestick charts
- Moving averages (MA20 / MA50)
- RSI indicator
- Volume analysis
- Automatic support & resistance detection
- AI-based next price prediction

## Live Demo
https://ai-trading-analytics-dashboard.streamlit.app
