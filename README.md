# AI Trading Analytics Dashboard

A Streamlit-based trading analytics dashboard that provides technical analysis and AI-based price prediction for stocks, forex, and crypto markets.

## Features

* Candlestick chart visualization
* Moving Averages (MA20 / MA50)
* RSI Indicator
* Volume Analysis
* Automatic Support & Resistance Detection
* AI Next Price Prediction (Linear Regression)
* Interactive charts using Plotly

## Tech Stack

* Python
* Streamlit
* yfinance API
* Plotly
* Scikit-learn
* Pandas / NumPy

## How to Run

Install dependencies:

pip install streamlit yfinance plotly scikit-learn numpy pandas

Run the dashboard:

streamlit run dashboard.py
